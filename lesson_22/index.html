Задача №3

Що потрапить в консоль?

if (2 * 20 <= 10 || 30 / 2 < 5 && 10 <= "10" || 20 === "20") {

console.log('000');

}

В консоль не потрапить нічого. Якщо розбирати по діям: 1. 2*20 = 40, 40 <= 10 - це false; 2. 30 / 2 = 15, 15 < 5 - це false; 10 <= "10" - це true, так як цей оператор порівняння може перевести з рядка в число, але умова логічного оператора "&&" оби обидва значення повертали true, тому умова не виконана і загальний вираз все одно повертає false; 3. 20 === "20" - false, тому що оператори суворого порівняння не змінюють тип даних. Отже, умова для виведення числа в консоль не виконується.

Задача №4

Створіть функцію, яка повертає результат ділення числа a на число b з додаванням рядка "Результат ділення: "

Викличте функцію передаючі різні значення, у тому числі не передаючи зовсім. Функція не має повертати NaN, Infinite або помилку.

function division(a = 1, b = 1) {

if ( typeof a !== 'number' || typeof b !== 'number') {

return "Аргументи мають бути числами"

} else if ( b === 0) {

return "Ділення на нуль неможливе"

} else {

let result = a / b

return `Результат ділення: ${result}`

}

}
